# GlobalChat

A chat application built using React With TypeScript and a dedicated backend to manage handle messaging.

When a user signs up, he is already added to a group called Global Chat. The user can see then start with that group, and make friends. He/She can add someone to their contacts by knowing their usernames.

A user can also create a group of his own with some other members.

It uses websocket to handle real-time communication.

It also supports Google-Login to ease the login process.

The application is designed to be user-friendly and responsive, making it easy to use on both desktop and mobile devices.
